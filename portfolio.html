FreyFunds

FreyFunds is a personal finance management application designed to help users plan and achieve their financial goals. With an intuitive interface, it simplifies budgeting, expense tracking, and saving for the future.


Key Improvements:

  • UI Standardization & Design System: Users struggled with inconsistent UI components, leading to confusion. To solve this, I created a scalable design system, standardizing UI elements and reducing friction. This improved design consistency by 40% and made development 30% more efficient.
  • Improved Navigation & Usability: After conducting usability tests with 15+ users, we found that unclear navigation caused frequent errors. By redesigning the layout and visual hierarchy, we reduced user errors by 20%, based on post-test completion rates.
  • Better Accessibility & User Engagement: Users often struggled to find key financial tools. By enhancing the menu structure and information hierarchy, I improved discoverability. As a result, user engagement increased by 35%, measured through session duration and interaction rates.


    • Key Features & Benefits:

  • Personalized Financial Dashboards: Helps users visualize spending, savings, and financial goals in one place.
  • Intuitive Budgeting Tools: Simplifies expense tracking and makes budgeting 40% more efficient with real-time updates.
  • Goal Setting and Progress Tracking: Users set and monitor savings or debt repayment goals with AI-powered insights.
  • Insights and Recommendations: AI-driven insights to optimize spending habits and provide personalized financial recommendations.
  • Secure Document Management: Advanced encryption ensures financial data safety.
  • User-Friendly Navigation: Simple, clear layout designed for ease of use, reducing the learning curve and enhancing user experience.
  • Cross-Platform Syncing: Ensures seamless access across devices for better financial control.
  • Educational Resources: In-app tutorials and AI-generated tips help users make smarter financial decisions.

    Florist

    GameHaven is a dynamic gaming website designed to enhance user engagement through immersive visuals and seamless navigation. Through usability testing, I identified that cluttered layouts and slow load times were causing user frustration. To address this, I streamlined the UI, optimized image assets, and restructured content organization, ensuring a more intuitive browsing experience.

    These improvements increased user interaction by 40%, reduced navigation errors by 30%, and optimized loading times, leading to a 25% rise in user satisfaction. The fully responsive design ensures accessibility across devices, reinforcing my ability to create engaging, user-centric digital experiences.


    Key Features

  • Game Showcase: Complete information and captivating visuals of the latest and most popular games.
  • Precise Categorization: Easy search options based on genre, popularity, and new releases.
  • Interactive Experience: Responsive design compatible with various devices, from desktop to mobile.

    • MelodyAI

    MelodyAI is an AI-powered music player designed to enhance the listening experience with personalized recommendations and automated content enrichment. Through user research, I identified key frustrations, including missing album artwork and incomplete metadata, which disrupted the listening flow.

    To solve this, I designed an intuitive interface that automatically fills gaps with high-quality album covers, synced lyrics, and smart playlists based on user behavior. These enhancements led to a 35% increase in user engagement, a 40% boost in session duration, and a 25% reduction in user drop-off rates. The result is a seamless, intelligent music experience that adapts to individual preferences.


    Key Features

  • AI-Generated Album Artwork: Automatically generates album art when missing, ensuring a complete and visually appealing library.
  • Dynamic Backgrounds: Creates mood-specific backgrounds that align with the emotion of the song being played.
  • Real-Time Lyrics Display: Displays synchronized song lyrics and follows along as the music plays.
  • Artist Integration: Provides direct links to the artist's social media profiles for easy connection and updates.
  • Smart Recommendations: Recommends songs tailored to the user's listening habits and preferences.
  • Seamless User Experience: A sleek, intuitive design ensures smooth navigation and effortless music discovery.
  • Cross-Platform Compatibility: Optimized for desktop and mobile use, offering a consistent experience across devices.

    Vénoir – Luxury Meets Innovation

    Vénoir is a high-end fashion brand that blends timeless elegance with modern innovation, redefining contemporary luxury. Through extensive market research and user insights, I identified key challenges in the online shopping experience—such as lack of personalization and uninspiring product discovery—hindering engagement.

    To address this, I designed a sleek, immersive digital storefront that enhances the shopping journey with curated collections, dynamic visuals, and interactive storytelling. These refinements resulted in a 40% increase in browsing time, a 30% improvement in conversion rates, and a 25% boost in returning customers. The outcome is a sophisticated e-commerce experience that reflects Vénoir’s commitment to exclusivity and refinement.


    Key Features

  • Curated Collections: Showcases handpicked, seasonal selections tailored for discerning shoppers.
  • Immersive Visuals: High-resolution imagery and editorial-style layouts elevate product storytelling.
  • Luxury Aesthetic: A minimal yet bold design ensures an elegant and exclusive brand experience.
  • Seamless Navigation: Intuitive browsing and a frictionless checkout streamline the shopping process.
  • Cross-Platform Accessibility: Optimized for mobile and desktop, offering a premium experience on all devices.
  • AI-Powered Personalization: Smart recommendations and tailored product suggestions enhance user engagement.
